# Mental Age Test

> **Mental age quiz** — Discover your mental age through a personality-style quiz. Next.js + Tailwind, mobile-friendly.

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Next.js](https://img.shields.io/badge/Next.js-black?logo=next.js)](https://nextjs.org/)

---

## Overview

A quiz-style web app that estimates your "mental age" — how mature, youthful, or seasoned your responses suggest you are — based on a series of personality and lifestyle questions. Built with Next.js and Tailwind CSS, optimized for mobile sharing.

A sister app to [braintype](https://github.com/moony01/braintype) and the [kpopface](https://github.com/moony01/kpopface) family, hosted under the moony01 entertainment-tests umbrella.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| **Framework** | Next.js |
| **Language** | TypeScript |
| **Styling** | Tailwind CSS |

## Local Development

```bash
git clone https://github.com/moony01/mentalage.git
cd mentalage

npm install
npm run dev
```

Open http://localhost:3000/
